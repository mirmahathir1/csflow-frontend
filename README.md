# csflow-frontend

March, 2021 - July, 2021

Node.js, Heroku, Vue.js, Firebase Hosting, MySQL, ClearDB Hosting, GCP Storage

This is an academic platform as our final software development project for the students of the department of CSE of BUET to share and store academic questions, answers and study materials including theses and projects.

For more information: Backend Repository (https://github.com/mirmahathir1/csflow-backend), Frontend Repository (https://github.com/mirmahathir1/csflow-frontend), Website (https://csflow-buet.web.app/#/), Design (https://www.figma.com/proto/cRS7AN9VBlOABQGLqH6SC3/CSFlow?node-id=2-3&starting-point-node-id=2%3A3&scaling=scale-down)
